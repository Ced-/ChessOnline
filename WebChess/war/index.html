ONLINE CHESS

Bitte den Namen der Gamesession eingeben, der Sie beitreten möchten
 (bzw. unter dem Sie eine neue Session starten möchten)


Zum Spielen müssen 2 Spieler einen beliebig wählbaren Namen für eine Spielesession eingeben (beide Spiele müssen denselben Namen wählen um innerhalb derselben Session gegeneinander spielen zu können).

Als kleines Bonusfeature können beliebig viele weitere Gäste den Spielverlauf live beobachten, indem sie der Spielsession durch Eingabe des entsprechenden Namens ebenfalls beitreten (aktiv spielend sind allerdings immer nur die ersten Beiden, die eine Spielsession betreten).

Es können natürlich beliebig viele Spielsessions parallel gestartet werden, ohne, dass diese sich in irgendeiner Art beeinflussen würden. Eine neue Spielsession wird immer automatisch gestartet, sobald man einen Namen für eine Spielsession wählt, unter der aktuell kein anderes Spiel stattfindet.
